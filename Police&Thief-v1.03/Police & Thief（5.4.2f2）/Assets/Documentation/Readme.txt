Endless Runner Template v1.0

GETTING STARTED
---------------
1. Import the unity package into an empty project.
2. In the Project tab, Go to Assets > Scenes > Scene_1.
  -- Observe: A dialogue box will appear asking if you would like to save
the current scene.
4. Click "Don't Save".

Note: For detailed instructions refer to the user documentation.
